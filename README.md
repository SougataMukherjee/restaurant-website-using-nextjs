## Getting Installed

First, run the terminal:

```bash
npx create-next-app@14.1.3 ./ or npx create-next-app@latest
npm i react-icons



```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev



```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
